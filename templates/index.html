Simulate Workload ▸

Mixed CPU + memory + disk + GPU load, varied over time to look like a real app: thread counts and held memory cycle through slices, disk + GPU alternate bursts with idles. Use to validate thermal, PSU and reconciler behaviour under sustained mixed pressure. One process per subsystem (and per GPU) - outputs stream side by side.
